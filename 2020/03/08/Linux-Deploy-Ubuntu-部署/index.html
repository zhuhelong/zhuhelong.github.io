手机安装 Linux Deploy

手上刚好有一台闲置的手机，把它刷成了魔趣，闲着也是闲着，耍起来~

项目 型号 备注
手机 LEX820
Android 8.1.0
魔趣 MK81.0-x2-200215-HISTORY
Kernel 3.18.109-mokee-gae88e076ad8
Linux Deploy 2.6.0-259
Ubuntu 18.04 LTS

前提：

  1. 手机需要 root，我使用的是 Magisk Manager，在魔趣的刷机快速上手中有介绍如何安装。
  2. 安装好 Linux Deploy、BusyBox（Linux Deploy 官方建议安装）、选装 Hacker's Keyboard、JuiceSSH（个人认为 Android 下最好用的 SSH 终端）。

这些软件都能在 Google Play 商店中找到，所以强烈建议安装 Google Play 商店，在魔趣中可以直接刷入 OpenGapps 。

安装 Ubuntu

Linux Deploy 中安装 Ubuntu 非常简单。

创建并选择配置文件

右上角打开菜单，打开配置文件选项，新建一个配置文件，我安装的是 Ubuntu，所以直接将配置文件命名为 Ubuntu。

设置配置文件属性

从上一步返回到 Linux Deploy 的主界面，点击右下角的设置按钮。我的是按照如下配置的。

这里有几点要注意，其中架构可以点击主界面的右上角的信息菜单中的状态选项查看，当然，一般 Android 的机器都是 arm 架构。发行版选择 bionic 为最新。源地址要特别注意，选择国内源，我选的是清华源：http://mirrors.tuna.tsinghua.edu.cn/ubuntu-ports/ 。安装类型之后的选项请参考官网自己做选择。请一定勾选启用 SSH 服务器，不然无法和 Linux 交互。

安装部署软件

安装 nginx

sudo apt install nginx

之后启动

sudo service nginx start

注：Linux Deploy 无法使用 systemctl，详情请了解 chroot。

安装 jdk

官网下载 arm 架构的 jdk，我使用的是 jdk 1.8，注意后缀一定要包含 linux-arm32 或 linux-arm64，32 还是 64 取决于你的手机 CPU 架构。

update-alternatives 连接一下

然后写个 Hello World 测试一下。
